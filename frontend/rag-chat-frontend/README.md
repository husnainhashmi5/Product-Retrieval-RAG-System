# Product Retrieval Frontend

React + Vite UI for the Product Retrieval RAG API.

## Commands

```bash
npm install
npm run dev
npm run lint
npm test
npm run build
```

Set `VITE_API_BASE_URL` when the backend is not running at `http://localhost:8000`.

The UI renders product cards from the structured API response. It does not parse products from generated assistant text.
